
## Templates

下标一定要从 $1$ 开始.

时间复杂度 $O(\log n)$.

单点修改, 范围查询.

```java
static class FenwickTree {
    private final long[] tree;
    private final int n;

    public FenwickTree(int n) {
        tree = new long[n + 1];
        this.n = n;
    }

    public void add(int i, long val) {
        while (i <= n) {
	        tree[i] += val;
	        i += i & -i;
        }
    }

    public long pre(int i) {
        long res = 0;
		while (i > 0) {
			res += tree[i];
			i -= i & -i;
		}
        return res;
    }

    public long query(int l, int r) {
        if (r < l) {
            return 0;
        }
        return pre(r) - pre(l - 1);
    }
}
```
