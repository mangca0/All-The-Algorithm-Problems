## 二、二叉树
### §2.6 二叉树的直径
*<span style="color:#6b7280">类似[[树形DP]]中的方法一：枚举「拐弯」点，最长+次长</span>*

在当前节点「拐弯」的直径长度 = 左子树的最长链 + 右子树的最长链 + 2
返回给父节点的是以==当前节点为根的子树的最长链== = max(左子树的最长链, 右子树的最长链) + 1
```java
class Solution {
    private int ans = 0;

    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);
        return ans;
    }

    private int dfs(TreeNode node) {
        if (node == null)
            return -1;
        int l = dfs(node.left), r = dfs(node.right);
        ans = Math.max(ans, l + r + 2);
        return Math.max(l, r) + 1;
    }
}
```
也可以
```java
	private int dfs(TreeNode node) {
        if (node == null)
            return -1;
        int l = dfs(node.left) + 1, r = dfs(node.right) + 1;
        ans = Math.max(ans, l + r);
        return Math.max(l, r);
    }
```
这是[543. 二叉树的直径](https://leetcode.cn/problems/diameter-of-binary-tree/)

---
有关树上路径问题，通常可以「枚举拐点」，整合左右信息更新答案，并向上返回一侧可延伸值
以上可看为链长的最长的路径
[687. 最长同值路径](https://leetcode.cn/problems/longest-univalue-path/)（值相同才延伸）
[124. 二叉树中的最大路径和](https://leetcode.cn/problems/binary-tree-maximum-path-sum/)（负数截断为0）
两题都可看为同一后序 DFS 框架，加上不同度量与约束
