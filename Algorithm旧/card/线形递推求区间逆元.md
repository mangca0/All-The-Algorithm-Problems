### 不同与费马小定理需要p为质数，此方法不限定p

设$p = k \cdot i + r$
有$k \cdot i + r \equiv 0 (modp)$
$k \cdot r^{-1} + i^{-1} \equiv 0 (modp)$
$i^{-1} \equiv -k + r^{-1}$
$i^{-1} \equiv - \lfloor \frac{p}{i} \rfloor +( p \% i)^{-1} (mod p)$

O(n)
```
inv[1] = 1;
for(int i = 2; i <= n; i ++)
	inv[i] = (p - p / i) * inv[p % i] % p;
```
