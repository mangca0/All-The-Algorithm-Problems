**质因数分解**
```
vector<pair<int, int>> v; //{num, cnt}
for(int i = 2; i * i <= x; i ++)
{
	int cnt = 0;
	if(x % i)continue;
	// 从小到大枚举，能整除的一定是质因子
	// 因为从2，3开始，合数一定会被其之前的质数率先分解掉
	while(x % i == 0)cnt ++, x /= i; //一直除到干净
	v.push_back({i, cnt});
}
if(x > 1)v.push_back({x, 1}); //剩一个大质数，即此时的x
for(const auto &i : v)cout << i.first << ' ' << i.second << endl;
```

# 约数和定理
通过[[唯一分解定理]]分解后，约数和 = $(p1^{0} + p1^{1} + ... + p1^{k1}) \cdot (p2^{0} + p2^{1} + ... + p2^{k2}) \cdot ... \cdot (pm^{0} + pm{1} + ... + pm^{km})$
