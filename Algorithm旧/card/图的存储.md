邻接表
g\[u]存储的是点 𝑢 的所有出边的相关信息（终点、边权等）。
```cpp
vector<pair<int, int>> g[n + 1];
```

```java
// jdk8需要自定义类
class Pair
{
	int v;
	int w;
	Pair(int v, int w){this.v = v; this.w = w}
}
// 需要手动new
List<Pair>[] g = new ArrayList[n + 1];
// 二选一
for(int i = 1; i <= n; i ++)g[i] = new ArrayList<>();
Arrays.setAll(g, i -> new ArrayList<>());
```
