# 直径
### 跑两次dfs法
**可求出直径端点编号，以及依赖直径端点维护的基础信息**
```cpp
// 如果根结点深度定义为0， 则需令dep1[0] = depu[0] = depv[0] = -1
inline void dfs(int x, int fa, int dep[])
{
	dep[x] = dep[fa] + 1;
	for(const auto &y : g[x])
	{
		if(y == fa)continue;
		dfs(y, x, dep);
	}
}
```
# 重心
mss为删除重心后，剩余联通快的最大值
### 性质
性质一：重心点mss值一定 <=n/2，非重心点一定 >n/2
性质二：重心至多只有2个，且2个一定连在一起，切断连接可分为2个大小一样的子树
性质三：树中所有点到重心的距离和最短，反之最短的是重心，用于处理最优化，最小代价问题
性质四：2个树连边，新重心在大树的连接点和原重心的简单路径上，若2个树大小相同，则重心为两个连接点

### 求mss
```cpp
inline void dfs(int x, int fa)
{
	siz[x] = 1;
	for(const auto &y : g[x])
	{
		if(y == fa)continue;
		dfs(y, x);
		siz[x] += siz[y];
		mss[x] = max(mss[x], siz[y]);// 子树联通块
	}
	mss[x] = max(mss[x], n - siz[x]);// 父亲联通块
}
```
