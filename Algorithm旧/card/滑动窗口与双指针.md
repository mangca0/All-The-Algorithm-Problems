## 一、定长滑动窗口
```
int l = 0;
for(int i = 1; i <= n; i ++)// 枚举右端点
{
	// 处理右端点
	sum += a[i];
		
	// 更新答案
	ans = max(ans ,sum);
	
	// 移动左端点，为下次右端点又移作准备
	sum -= a[l];
	l --;
}
```
## 二、不定长滑动窗口
最短、最长、方案数
==越短越合法/求最长/最大，惰性收缩
越长越合法/求最短/最小，积极收缩==
```
int l = 0;
for(int i = 1; i <= n; i ++)// 枚举右端点
{
	// 处理右端点
	sum += a[i];
	
	// 考虑条件，移动l
	while()
	{
		sum -= a[l];
		l ++;
	}
	
	// 更新答案
	ans = max(ans ,sum);
}
```
