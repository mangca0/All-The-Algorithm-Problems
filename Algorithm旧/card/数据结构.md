## 零、常用枚举技巧
### §0.3 遍历对角线
模板，按对角线升序降序排列
```java
class Solution {
    public int[][] sortMatrix(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        // k = i-j+n
        // k～[1, m+n-1]
        for(int k = 1; k < m + n; k ++)
        {
            // j = i-k+n
            int minJ = Math.max(n - k, 0); // i=0，j不能是负数
            int maxJ = Math.min(m - 1 - k + n, n - 1); // i=m-1，j不能超过n-1
            List<Integer> a = new ArrayList<>(maxJ - minJ + 1);
            for(int j = minJ; j <= maxJ; j ++)
                a.add(grid[k + j - n][j]);
            a.sort(minJ > 0 ? null : Comparator.reverseOrder()); // j<=0为主对角线及以下，null为默认升序
            for(int j = minJ; j <= maxJ; j ++)
                grid[k + j - n][j] = a.get(j - minJ);
        }
        return grid;
    }
}
```
模板，遍历副对角线

class Solution {
    public int[] findDiagonalOrder(int[][] mat) {
        int m = mat.length, n = mat[0].length;
        int[] ans = new int[m * n];
        int idx = 0;
        for (int k = 0; k < m + n - 1; k++) {
            int minJ = Math.max(k - m + 1, 0);
            int maxJ = Math.min(k, n - 1);
        }
        return ans;
    }
}
