## 1、朴素选择排序：
```
char str[10][100];//最多10个字符串，每个最多100个字符
char *strp[10];//存储每个字符串的首地址

//假设5个字符串
//读入
for(int i = 1; i <= 5; i ++)
    {
        scanf("%s", a[i]);
        b[i] = a[i];
    }
//选择排序
for(int i = 1; i <= 4; i ++)
    for(int j = i + 1; j <= 5; j ++)
        if(strcmp(b[i], b[j]) > 0)
        {
            char *tmp = b[i];
            b[i] = b[j];
            b[j] = tmp;
        }
//输出
for(int i = 1; i <= 5; i ++)printf("%s\n", b[i]);
```
