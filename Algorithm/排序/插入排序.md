假设当前待插入元素在 i 位置，那么 0~i-1 上已经有序，新来的数从右到左滑到不再小的位置插入，然后继续。

一个与插入排序相同的操作是打扑克牌时，从牌桌上抓一张牌，按牌面大小插到手牌后，再抓下一张牌。

数据范围较小时还可以用用。

```java
public static void insertionSort(int[] arr) {  
    // 0~0 位置已经有序，从 1 位置开始  
    for (int i = 1; i < arr.length; i++) {  
        // 0~i-1 已经有序，新来的数是 i        
        // 这样写更简洁，也不会慢多少  
        for (int j = i - 1; j >= 0 && arr[j] > arr[j + 1]; j--) {  
            swap(arr, j, j + 1);  
        }  
        // int key = arr[i];

        // }

    }  
}
```

插入排序是一种稳定的排序算法。

插入排序的最优时间复杂度为 $O(n)$，在数列几乎有序时效率很高。
插入排序的最坏时间复杂度和平均时间复杂度都为 $O(n^2)$。

## 折半插入排序
因为 0~i 范围上已经有序，可以通过二分算法优化性能，在排序元素数量较多时优化的效果比较明显。

折半插入排序与直接插入排序的基本思想是一致的，折半插入排序仅对插入排序时间复杂度中的常数进行了优化，所以优化后的时间复杂度仍然不变。

```java
public static void insertionSort(int[] arr) {  
    // 0~0 位置已经有序，从 1 位置开始  
    for (int i = 1; i < arr.length; i++) {  
        // 0~i-1 已经有序，新来的数是 i        
        int index = upperBound(arr, 0, i - 1, arr[i]); // 在 0~i-1 上找第一个 >arr[i] 的位置  
        for (int j = i - 1; j >= index; j--) {  
            swap(arr, j, j + 1);  
        }  
    }  
}  

// 左闭右开写法
private static int upperBound(int[] arr, int l, int r, int x) {  
    r++;  
    while (l < r) {  
        int mid = l + r >> 1;  
        if (arr[mid] > x) r = mid;  
        else l = mid + 1;  
    }  
    return r;  
}
```
