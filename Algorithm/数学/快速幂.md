将指数二进制拆解，从而优化幂运算，$O(\log b)$。

```java
long qpow(long a, int b) {
	long res = 1;
	while (b > 0) {
		if ((b & 1) == 1) res = res * a % MOD;
		a = a * a % MOD;
		b >>= 1;
	}
	return res;
}
```
