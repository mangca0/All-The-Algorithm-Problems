前置知识：网格图/二维 DP。

01 背包：有 $n$ 个物品，第 $i$ 个物品的价值为 $values[i]$，体积为 $costs[i]$，每个物品最多选择一次，$\begin{cases}至多装 capacity，求方案数/最大价值和。 \\恰好装 capacity，求方案数/最大/最小价值和。 \\至少装 capacity，求方案数/最小价值和。\end{cases}$ 

实现时，常用 **选或不选** 的思路。

时间复杂度一般都是 $O(物品个数 \cdot 背包容量)$。

## 模板和空间优化
二维 DP 中，**空间优化** 较为重要，尤其是一维数组版本（优化二）。

```python
f = [[False] * (s + 1) for _ in range(n + 1)]
f[0][0] = True
for i, x in enumerate(nums):
	for j in range(s + 1):
		f[i + 1][j] |= f[i][j]
		if j >= x:
			f[i + 1][j] |= f[i][j - x]
return f[n][s]
```
空间复杂度 $O(ns)$。

### 优化一
不难发现，在计算 $i+1$ 层时只会用到 $i$ 层的状态。所以我们可以只用两层数组，交替更新。

代码用取模实现：
```python
f = [[False] * (s + 1) for _ in range(2)]
f[0][0] = True
for i, x in enumerate(nums):
	for j in range(s + 1):
		f[(i + 1) % 2][j] |= f[i % 2][j]
		if j >= x:
			f[(i + 1) % 2][j] |= f[i % 2][j - x]
return f[n % 2][s]
```
空间复杂度 $O(s)$。

### 优化二
我们能不能彻底优化掉第一个维度，反复利用一个数组呢？

实际上，同一层的状态依赖是从左到右的，所以倒序遍历 $j$ 就能实现：
```python
f = [False] * (s + 1)
f[0] = True
for x in nums:
	for j in range(s, x - 1, -1):
		f[j] |= f[j - x]
return f[s]
```

```java
int[] f = new int[T + 10];
for (int i = 1; i <= M; i++) {
	for (int j = T; j >= costs[i]; j--) {
		f[j] = Math.max(f[j], f[j - costs[i]] + values[i]);
	}
}
out.println(f[T]);
```
这题是 [洛谷 P1048 采药](https://www.luogu.com.cn/problem/P1048)。

空间复杂度 $O(s)$。

### 优化三
$bitset$ 优化：状态依赖在数组上是从左到右的，反应到二进制数上就是从右到左，所以要将 $f$ 左移。
```python
f = 1
for x in nums:
	f |= f << x
return (f >> s & 1) == 1
```
空间复杂度 $O\left( \frac{s}{32} \right)$。

这题是 [力扣 416. 分割等和子集](https://leetcode.cn/problems/partition-equal-subset-sum/)。

## 循环顺序
在二维动态规划问题中，若要用一维空间优化，那么对于要计算的 $f[i][j]$ 的状态，若转移来源于第 $i$ 层、$j$ 左侧的状态，或来源于第 $i-1$ 层、$j$ 右侧的状态（即左下或右上），则循环顺序从左到右；
若转移来源于第 $i$ 层、$j$ 右侧的状态，或来源于第 $i-1$ 层、$j$ 左侧的状态（即右下或左上），则循环顺序从右到左。
