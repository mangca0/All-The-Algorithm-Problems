# 模板
```cpp
inline int find(int x)
{
	return fa[x] = (fa[x] == x ? x : find(fa[x]))
}

inline void merge(int x, int y)
{
	x = find(x), y = find(y);
	if(x == y)return;
	if(siz[x] < siz[y])swap(x, y);
	fa[y] = x;
	siz[x] += siz[y];
	sum[x] += siz[y];
}
```

# 优化操作
### 1.路径压缩 -> find
**无论fa的定义如何变，find永远返回x或find(fa[x])** [lanqiao 185 修改数组]
```
return fa[x] = (fa[x] == x ? x : find(fa[x]))
```
优点：查询几乎为o(1)
缺点：破坏树的结构
### 2.按秩排序 -> merge
秩：树深
矮树 -> 高树
```
//x->y
if(rank1[x] > rank1[y])swap(rootX, rootY);
fa[rootX] = rootY;

if(rank1[x] == rank1[y])rank[y] ++;
```
### 3.启发式排序 -> merge
小树 -> 大树
```
//x->y
if(siz[x] > siz[y])swap(rootX, rootY);
fa[rootX] = rootY, siz[y] += siz[x];
```
