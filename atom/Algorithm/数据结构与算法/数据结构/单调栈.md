最经典用法：单调栈可以在 $O(n)$ 时间内，求出每个位置左右两侧，离当前位置最近、且值严格小于（大于）的位置。

## 模板
以要求严格小于、单调栈递增为例。

下面是 **一次遍历** + **修正相同元素** + **出栈时记录答案** 写法：
```java
Deque<Integer> st = new ArrayDeque<>();
int[][] ans = new int[n + 10][2];

// 入单调栈
for (int i = 1; i <= n; i++) {
	while (!st.isEmpty() && a[st.peekLast()] >= a[i]) {
		int x = st.pollLast();
		ans[x][1] = i;
		ans[x][0] = st.isEmpty() ? -1 : st.peekLast();
	}
	st.offerLast(i);
}

// 清空单调栈
while (!st.isEmpty()) {
	int x = st.pollLast();
	ans[x][1] = -1;
	ans[x][0] = st.isEmpty() ? -1 : st.peekLast();
}

// 处理有重复值情况
for (int i = n; i >= 1; i--) {
	if (ans[i][1] != -1 && a[ans[i][1]] == a[i]) {
		ans[i][1] = ans[ans[i][1]][1];
	}
}
```

统一一套模板、一个遍历方向，思考量较小。

%% 下面是 **哨兵** + **正反两次遍历** + **进栈时记录答案** 写法：
```java
int[] left = new int[n + 10];
Deque<Integer> st = new ArrayDeque<>();
st.offerLast(-1); // 哨兵
for (int i = 1; i <= n; i++) {
	while (!st.isEmpty() && a[st.peekLast()] >= a[i]) {
		st.pollLast();
	}
	left[i] = st.peekLast();
	st.offerLast(i);
}

int[] right = new int[n + 10];
st.clear();
st.offerLast(-1); // 哨兵
for (int i = n; i >= 1; i--) {
	while (!st.isEmpty() && a[st.peekLast()] >= a[i]) {
		st.pollLast();
	}
	right[i] = st.peekLast();
	st.offerLast(i);
}
```

如果只需要 **单向** 的位置，正向或反向遍历会很方便，正向遍历可以确认 left 数组，反向确认 right。 %%
