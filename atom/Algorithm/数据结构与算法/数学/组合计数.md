# 基础
$C_n^m=\frac{n!}{(n-m)!\cdot m!}=\frac{n\cdot(n-1)\cdot...\cdot(n-m+1)}{m\cdot(m-1)\cdot...\cdot1}$
$C_n^0+C_n^1+...+C_n^n = 2^n$
$C_n^m=C_{n-1}^{m-1}+C_{n-1}^m$
$A_n^n=n!$

# DP法
$C_n^m=C_{n-1}^{m-1}+C_{n-1}^m$
### 理解
杨辉三角、是否选第一个
### 条件
n, m <= 3000，小范围，dp题，**可以对非质数取模**
### 写法
```
c[0][0] = 1;
for(int i = 1; i <= n; i ++)
{
	c[i][0] = 1;
	for(int j = 1; j <= m; j ++)
		c[i][j] = c[i - 1][j] + c[i - 1][j - 1];
}
```

# 题
[P2822 [NOIP2016 提高组] 组合数问题]
DP法 + 前缀和


## 模板
对于组合数有：
$$C_n^m=\frac{n!}{(n-m)!\cdot m!}$$

由于运用了费马小定理计算逆元，此计算方法**只能对质数取模**。
```java
// 预处理
public static void build() {
	// 预处理阶乘数组
	fac[0] = 1;  
	for (int i = 1; i < MX; i++) {  
	    fac[i] = fac[i - 1] * i % MOD;  
	}  
	// 预处理阶乘数组的逆元
	inv_fac[MX - 1] = qpow(fac[MX - 1], (int) MOD - 2);  
	for (int i = MX - 2; i >= 0; i--) {  
	    inv_fac[i] = inv_fac[i + 1] * (i + 1) % MOD;  
	}
}

// 快速幂
public static long qpow(long x, int p) {  
    long res = 1L;  
    while(p > 0) {  
        if ((p & 1) == 1) {  
            res = res * x % MOD;  
        }  
        x = x * x % MOD;  
        p >>= 1;  
    }  
    return res;  
}

// 组合数
public static long C(int n, int m) {  
    return m < 0 || m > n ? 0 : fac[n] * inv_fac[m] % MOD * inv_fac[n - m] % MOD;  
}
```
$O(N)$ 预处理 + $O(1)$ 查询。
