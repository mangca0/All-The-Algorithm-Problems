## 基础拷贝
```java
// 1.创建对象
FileInputStream fis = new FileInputStream(`name:`);
FileOutputStream fos = new FileOutputStream(`name:`);

// 2.拷贝：边读边写
int b;
while((b = fis.read()) != -1) {
	fos.write(b);
}

// 3.释放资源：先开的后关
fos.close();
fis.close();
```

上面方法 FileInputStream 一次读一个字节，非常慢。

## 改进拷贝
```java
FileInputStream fis = new FileInputStream(`name:`);
FileOutputStream fos = new FileOutputStream(`name:`);

int len; // 返回读取到的字节长度
byte[] bytes = new byte[1024 * 1024 * 5]; // 字节数组大小一般为字节的 1024 倍，这里是5M
while((len = fis.read(bytes)) != -1) {
	fos.write(bytes, 0, len);
}

fos.close();
fis.close();
```
