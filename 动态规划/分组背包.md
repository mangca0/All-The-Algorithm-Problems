在 01 背包 的基础上，多个物品分组，每组至多只能选一个物品。

$$dfs(i, j) = max(dfs(i - 1, j), dfs(i - 1, j - costs[k]) + values[k]$$

这里枚举 $k$ 用的是 **枚举选哪个** 的思想。

时间复杂度：$O(物品个数 \cdot 背包容量)$，对比 01 背包并不会升阶。
空间复杂度：$O(组个数 \cdot 背包容量)$。

## 模板和空间优化
记忆化搜索：
```java
public static int dfs(int i, int j) {  
	if (i < 1) return 0;  
	if (memo[i][j] != -1) return memo[i][j];  
	List<Integer> group = groups.get(i);  
	int res = dfs(i - 1, j);  
	for (int k : group) {  
		if (j - a[k][0] >= 0) res = Math.max(res, dfs(i - 1, j - a[k][0]) + a[k][1]);  
	}  
	return memo[i][j] = res;  
}  
out.println(dfs(cnt, m));
```
背包容量至多为 $m$，求最大价值和

二维递推：
```java
int[][] f1 = new int[cnt + 1][m + 1];  
for (int i = 1; i <= cnt; i++) { // cnt 为组数
    for (int j = 0; j <= m; j++) {  
        f1[i][j] = f1[i - 1][j];  
        for (int k : groups.get(i)) {  
            if (j - a[k][0] >= 0)  
                f1[i][j] = Math.max(f1[i][j], f1[i - 1][j - a[k][0]] + a[k][1]);  
        }  
    }  
}  
out.println(f1[cnt][m]);
```

一维递推：
```java
int[] f2 = new int[m + 1];  
for (int i = 1; i <= cnt; i++) { // cnt 为组数
    for (int j = m; j >= 0; j--) {  
        for (int k : groups.get(i)) {  
            if (j - a[k][0] >= 0)  
                f2[j] = Math.max(f2[j], f2[j - a[k][0]] + a[k][1]);  
        }  
    }  
}  
out.println(f2[m]);
```

或者这样遍历组内元素，可以省略预处理 $groups$：
```java
int[] f2 = new int[m + 1];  
for (int start = 1, end = 2; start <= n; ) {  
    while (end <= n && a[start][2] == a[end][2]) end++;  
    // start~end-1  
    for (int j = m; j >= 0; j--) {  
        for (int k = start; k < end; k++) {  
            if (j - a[k][0] >= 0)  
                f2[j] = Math.max(f2[j], f2[j - a[k][0]] + a[k][1]);  
        }  
    }  
    start = end++;  
}  
out.println(f2[m]);
```

## 至少/至多 变形模板
```python
m = len(mat)
# 至少为 target 的最小价值和
f = [inf] * (target + 1)
f[0] = 0
for i in range(m):
	for j in range(target, -1, -1):
		res = inf
		for k in mat[i]:
			res = min(res, f[max(0, j - k)] + k)
		f[j] = res
# print(f[-1])
ans = abs(f[target] - target)
if ans == 0:
	return ans

# 至多为 target 的最大价值和
f = [0] * (target + 1)
for i in range(m):
	for j in range(target, -1, -1):
		res = -inf
		for k in mat[i]:
			if j - k >= 0:
				res = max(res, f[j - k] + k)
		f[j] = res
# print(f[-1])
ans = min(ans, abs(f[target] - target))
return ans
```
这题是 [力扣 1981. 最小化目标值与所选元素的差](https://leetcode.cn/problems/minimize-the-difference-between-target-and-chosen-elements/)。
