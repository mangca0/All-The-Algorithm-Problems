[983. 最低票价](https://leetcode.cn/problems/minimum-cost-for-tickets/)
这题可以看成「爬楼梯」题目的变种。
有两种思考角度，每种角度有两种写法。

## 角度一
我们从旅游的第一天 $i$ 开始思考，$n$ 为旅行的最后一天，寻找子问题，分类讨论：
- 在第 $i$ 天购买 $1$ 天的车票，接下来要解决：从 $i+1$ 到 $n$ 天的最小花费
- 在第 $i$ 天购买 $7$ 天的车票，接下来要解决：从 $i+7$ 到 $n$ 天的最小花费
- 在第 $i$ 天购买 $30$ 天的车票，接下来要解决：从 $i+30$ 到 $n$ 天的最小花费
这和「爬楼梯」选择爬几层的思路很像。

### 写法一
本题数据范围很小，$n≤365$，故可以直接以自然日为下标，用动态规划在自然日上推进。
定义 $dfs(i)$ 表示 $i$ 到 $n$ 天的最小花费。
- 若第 $i$ 天不是旅游日，就有：
$$dfs(i) = dfs(i + 1)$$
- 若第 $i$ 天是旅游日，则有：
$$dfs(i) = min(dfs(i+1)+costs[0], dfs(i+7)+costs[1], dfs(i+30)+costs[2])$$
- 递归边界：当$i>n$ 时 $dfs(i)=0$，此时没有要旅行的天数
- 递归入口：$dfs(D_{first})$，其中 $D_{first}$ 为旅行的第一天
当然也可以直接把旅行第一天定为 $1$，最后一天定为 $365$。

加上记忆化就有：
```java
class Solution {
    boolean[] vis = new boolean[366];
    int[] cache = new int[366];
    int[] costs;

    public int mincostTickets(int[] days, int[] costs) {
        this.costs = costs;
        for (int x : days)
            vis[x] = true;
        Arrays.fill(cache, -1);
        return dfs(1);
    }

    private int dfs(int i) {
        if (i > 365)
            return 0;
        if (cache[i] != -1)
            return cache[i];
        if (vis[i] != true)
            return cache[i] = dfs(i + 1);
        int c1 = dfs(i + 1) + costs[0];
        int c2 = dfs(i + 7) + costs[1];
        int c3 = dfs(i + 30) + costs[2];
        int res = Math.min(c1, Math.min(c2, c3));
        cache[i] = res;
        return res;
    }
}
```
时间复杂度：$O(365)$ 或 $O(D_{last}-D_{first})$。
空间复杂度：$O(365)$ 或 $O(D_{last}-D_{first})$。

递推形式：
```java
class Solution {
    boolean[] vis = new boolean[366];
    public int mincostTickets(int[] days, int[] costs) {
        for (int x : days)
            vis[x] = true;
        int[] f = new int[370];
        f[366] = 0;
        for (int i = 365; i >= 1; i--) {
            if (vis[i] != true) {
                f[i] = f[i + 1];
                continue;
            }
            int c1 = f[Math.min(i + 1, 366)] + costs[0];
            int c2 = f[Math.min(i + 7, 366)] + costs[1];
            int c3 = f[Math.min(i + 30, 366)] + costs[2];
            f[i] = Math.min(c1, Math.min(c2, c3));
        }
        return f[1];
    }
}
```

### 写法二
当 $days[i]$ 比较大时，比如 $≤10^{9}$ 时，上面以值域推进的做法就不行了。
此时用**日期索引**上做「跳跃」，就能解决复杂度与 $days[i]$ 值域有关的问题了。

大体思路是一样的，我们定义 $dfs(i)$ 为第 $days[i]$ 天到第 $days[days.length-1]$ 的旅行最小花费。只需要在「跳跃」时，跳跃到第一个 $≥days[i]+(1,7,30)$ 的索引 $j$ 就行，这可以用二分优化。

记忆化搜索：
```java
class Solution {
    int[] cache, costs, days;
    int n;

    public int mincostTickets(int[] days, int[] costs) {
        this.costs = costs;
        this.n = days.length;
        this.days = days;
        cache = new int[n + 10];
        Arrays.fill(cache, -1);
        return dfs(0);
    }

    private int dfs(int i) {
        if (i >= n)
            return 0;
        if (cache[i] != -1)
            return cache[i];
        int c1 = dfs(lowerBound(i, 1)) + costs[0];
        int c2 = dfs(lowerBound(i, 7)) + costs[1];
        int c3 = dfs(lowerBound(i, 30)) + costs[2];
        int res = Math.min(c1, Math.min(c2, c3));
        cache[i] = res;
        return res;
    }

    // 左闭右开
    private int lowerBound(int i, int day) {
        int l = i + 1, r = n;
        while (l < r) {
            int mid = l + r >> 1;
            if (days[mid] >= days[i] + day)
                r = mid;
            else
                l = mid + 1;
        }
        return r;
    }
}
```
时间复杂度：$O(nlogn)$，其中 $n$ 为 $days$ 的长度。
空间复杂度：$O(n)$。

## 角度二
上面我们是从第一天开始入题，从左往右思考。我们也可以从右往左思考。
我们从旅游的最后一天 $i$ 开始思考，寻找子问题，分类讨论：
- 在第 $i$ 天购买 $1$ 天的车票，接下来要解决：从 $1$ 到 $i-1$ 天的最小花费
- 在第 $i-7+1$ 天购买 $7$ 天的车票，接下来要解决：从 $1$ 到 $i-7$ 天的最小花费
- 在第 $i-30+1$ 天购买 $30$ 天的车票，接下来要解决：从 $1$ 到 $i-30$ 天的最小花费
这种方法更方便把递归翻译成递推。

定义 $dfs(i)$ 为第 $1$ 到 $i$ 天的最小花费，后续思考大同小异，不再赘述。给出以值域推进的代码：
```java
class Solution {
    boolean[] vis = new boolean[366];
    int[] cache = new int[366];
    int[] costs;

    public int mincostTickets(int[] days, int[] costs) {
        this.costs = costs;
        for (int x : days)
            vis[x] = true;
        Arrays.fill(cache, -1);
        return dfs(365);
    }

    private int dfs(int i) {
        if (i <= 0)
            return 0;
        if (cache[i] != -1)
            return cache[i];
        if (vis[i] != true)
            return cache[i] = dfs(i - 1);
        int c1 = dfs(i - 1) + costs[0];
        int c2 = dfs(i - 7) + costs[1];
        int c3 = dfs(i - 30) + costs[2];
        int res = Math.min(c1, Math.min(c2, c3));
        cache[i] = res;
        return res;
    }
}
```
