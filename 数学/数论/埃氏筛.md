预处理 $[1,n]$ 的所有质数.

`isPrime` 为 `true` 为质数.

```java
public static boolean[] xxx (int n) {
	boolean[] isPrime = new boolean[n + 1];
	Arrays.fill(isPrime, true);
	isPrime[0] = isPrime[1] = false;
	for (int i = 2; i * i <= n; i++) {
		if (isPrime[i]) {
			for (long j = (long) i * i; j <= n; j += i) {
				isPrime[(int) j] = false;
			}
		}
	}
	return isPrime;
}
```

先枚举底数, 再筛掉其倍数的思想.
